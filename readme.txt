Acontinuacion se detalla las instrucciones de ejecucion del proyecto postman para el ejercicio de Api DemoBlaze:

1. Cargar(Importar) los dos archivos .json:
	API_DemoBlaze.postman_collection.json  -> Coleccion
	UAT.postman_environment.json   -> Variables de entorno

2. Selecionar el entorno(UAT).

3. Ejecutar las peticiones individual o de forma grupal con la opcion de <<Run Collection>>.

4. Validar los resultados de las peticiones y los tests configurados.
